# 1 HTTPS

- Hyper Text Transfer Protocol Secure
- HTTPS를 사용하면 모든 HTTP 요청과 응답 데이터는 네트워크로 보내지기 전에 암호화된다.
- HTTPS는 HTTP 하부에 전송 레벨 암호 보안 계층을 제공함으로써 동작한다.
  - 이 보안 계층은 안전 소켓 계층(SSL) 혹은 전송 계층 보안(TLS)을 이용해 구현된다.
  -  [TLS.md](../../TLS/TLS.md) 참고




# 2 HTTPS의 기능
