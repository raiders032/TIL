Description

"너희는 먹고 싶은 것만 골라, 나머진 내가 알아서 할게"


모믈렛을 통해 더 이상 식당을 찾고, 물어보고, 의견 모으고 할 필요 없이 쉽고 간편하게 식당을 결정하세요


• 친구, 주변 사람, 연인과 어떤 식당에서 먹을지 고민할 때, 식당좋아요 게임!

- 식당좋아요 게임은 내설정의 카테고리에 맞춰서 랜덤하게 식당을 보여줍니다

- 가고 싶으면 왼쪽으로 스와이프, 별로면 오른쪽으로 스와이프, 그저그러면 아래쪽으로 스와이프하세요

- 모믈렛이 자동으로 의견을 모아서 갈 곳을 정해줍니다!


• 주변에 어떤 식당이 있을까? 식당카드를 슥슥 넘기며 바로 확인하세요!


• 내가 먹고 싶은 것만 보고 싶은데... 내설정에서 식당 카테고리를 설정하세요!


• 여긴 나중에 가봐야지~ 사진 오른쪽 밑의 하트를 눌러 나만의 즐겨찾기에 저.장.


• 식당 정보가 잘못되었다면 혹은 내용을 추가하고 싶으면 사진 오른쪽 위의 버튼을 눌러 정보수정을 요청해주세요!


모믈렛을 통해 즐거운 식사시간 되세요 ~,~

(현재 서울 지역에서만 서비스 가능합니다)
